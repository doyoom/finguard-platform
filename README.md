# finguard-platform
Cloud-native financial risk scoring API platform built with Terraform, AWS EKS, Helm, GitHub Actions, and ArgoCD.

## Project Background

This project originated from the `cloudflix-infra` Terraform workspace developed during the AIM AWS Study.

Original reference:
https://github.com/AIM-AWS-study/aws-study/tree/main/final-project/cloudflix-infra

The infrastructure history was imported into this repository and later reorganized into a personal platform project focused on financial transaction risk scoring.
